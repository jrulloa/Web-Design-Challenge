SUMAMARY: Latitude vs. Variables

This project aims to analyze the change that the climate has as you get closer to the equator. We extracted data from the OpenWeatherMap API from a dataset in more than 500 cities to perform this analysis.

We use Matplotlib to identify various aspects of weather versus latitude. We used the factors: temperature, cloud cover, wind speed, and humidity. Here you will find the source data and the graphs created in the analysis and their descriptions, trends, correlations, and explanations of what was observed.
